CHANNELS
========
## start
```bash
# copy
$ git clone git@github.com:trabur/istrav-channels.git

# get dependancies
$ mix deps.get

# run phoenix server
$ mix phx.server
```
> Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.

## hosting
gigalixir.com:
> checkout https://istrav.gigalixirapp.com/


## init
bootstrap from phoenix website:
```bash
$ mix archive.install hex phx_new
$ mix phx.new demo --live
```
